# Тут знаходяться работи з якими я працював:


## Features

  - Resize, crop, pad, rotate, show and save images
  - Create image reflection
  - Crop whitespace
  - Sharpen images
  - Watermark



## Requirements

  - PHP >= 5.6
  - GD 2.0+
